One-Time Setup

This app stores everyone's brackets on JSONBin.io — a free cloud database that needs no coding. Complete this setup once, then share the URL with your colleagues. Takes about 3 minutes.

📋 Step-by-step

  1. Go to jsonbin.io and sign up free
  2. Click "+ New Bin", paste this exactly into the editor:
    {"users":[],"results":{"groups":{},"knockout":{}}}
    Name it WC2026-Shared and click Save
  3. After saving, copy the BIN ID shown (e.g. 664ab12c...)
  4. Go to Account → API Keys, copy your X-Master-Key
  5. Paste both below and click Save

Your key is stored only in your browser. It is sent only to JSONBin's servers.

How to Play
Everything you need to know
🔧
First-Time Setup (Organiser Only)

1. Sign up free at jsonbin.io
2. Create a new bin with the starter JSON
3. Copy your Bin ID and API Key
4. Click ⚙ Config and paste them in
5. Host this file on GitHub Pages
6. Share the URL — done!

👤
For Participants

1. Open the shared URL
2. Enter your name
3. Fill in Group Stage predictions
4. Fill in Knockout picks
5. Hit Save — you're in the competition!

🏆
Scoring System

Group positions: 1 pt each
Round of 32: 2 pts/pick
Round of 16: 3 pts/pick
Quarterfinals: 5 pts/pick
Semifinals: 8 pts/pick
